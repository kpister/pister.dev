**CS 538: Theory and Design of Programming Languages** Kaiser Pister A UW Madison Computer Science course exploring an introduction to programming language theory and design. [Course Syllabus gdoc](https://docs.google.com/document/d/1fHPlwfnbsCGGVy0ex1bVEI_nOpSk6yTzBYt2Vnw9zpw/edit?usp=sharing) [Course Syllabus pdf](538/syllabus.pdf) Work load ============================================================== There will be a written homework and a programming assignment due each week. Expectations ============================================================== You do not need to have any prior experience with the languages used in this course, we will cover the relevant aspects of each.
